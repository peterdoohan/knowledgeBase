## Key evidence

- Dorsomedial striatum is the proposed locus of SR-based value computation; lesions here impair reward devaluation sensitivity, consistent with the SR-TD model ([Russek 2017](../../raw/summaries/russek2017_model_based_reinforcement.md))
- The dorsomedial striatum and dorsolateral striatum implement different input representations (SR vs. punctate states) while sharing the same TD learning mechanism, providing a unified neural implementation where both model-based and model-free behaviors arise from the same circuitry ([Russek 2017](../../raw/summaries/russek2017_model_based_reinforcement.md))
- Ventral striatum (nucleus accumbens) provides pre-decision reward evaluation signals during VTE, representing the evaluation component of deliberative decision-making ([Redish 2016](../../raw/summaries/redish2016_vicarious_trial_error.md))
