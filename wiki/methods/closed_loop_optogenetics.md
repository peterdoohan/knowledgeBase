## Key evidence

- Closed-loop optogenetic disruption of content-specific hippocampal reactivation during sleep selectively impairs recall of the disrupted memories while leaving other memories intact ([Gridchyn2020](../../raw/summaries/gridchyn2020_replay_selective_memory.md)).

- Bayesian population-vector decoding can classify high synchrony events as target or control environment within ~1 ms latency, enabling real-time closed-loop control ([Gridchyn2020](../../raw/summaries/gridchyn2020_replay_selective_memory.md)).

- Content-specific memory consolidation can be selectively modulated using closed-loop brain-machine interface methods, demonstrating causal role of replay content ([Gridchyn2020](../../raw/summaries/gridchyn2020_replay_selective_memory.md)).
