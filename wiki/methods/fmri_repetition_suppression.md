# fMRI Repetition Suppression

fMRI repetition suppression is a neuroimaging analysis technique that leverages the reduced BOLD response to repeated stimuli to infer representational similarity, with greater suppression indicating that neural populations treat stimuli as similar.

## Key evidence

- fMRI repetition suppression reveals abstract relational map emergence in mPFC across consolidation. A large mPFC cluster (peak t21 = 5.1, MNI [2, 42, 8]; FWE p = 0.003) showed cross-graph suppression that was significantly greater in session 2 than session 1 (SVC FWE p < 0.05, t21 = 3.66), indicating emergence of abstract representation via consolidation ([Baram 2024](../../raw/summaries/baram2024_abstract_relational_map_consolidation.md))

- fMRI repetition suppression shows MTL maintains concrete graph representations across time. Distance-dependent suppression for both relevant and irrelevant graphs in left MTL (peak t22 = 4.39, MNI [-24, -22, -28]), with no difference between sessions or graph relevance (2×2 ANOVA, all p > 0.2), indicating concrete stimulus-specific representations ([Baram 2024](../../raw/summaries/baram2024_abstract_relational_map_consolidation.md))
