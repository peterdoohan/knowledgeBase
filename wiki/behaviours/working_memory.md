## Key evidence

- Human working memory maintenance involves persistent activity in both MTL concept cells and medial frontal cortex maintenance neurons, with persistent activity magnitude predicting behavioral accuracy and decreasing with WM load ([Kubska2021](../../raw/summaries/kubska2021_single_neuron_cognition.md)).

- Working memory content in humans can be read out from the phase of slow oscillations at which neurons fire, enabling multiplexing of multiple items across oscillatory cycles through combined phase-rate coding ([Kubska2021](../../raw/summaries/kubska2021_single_neuron_cognition.md)).

- Hippocampal-prefrontal coupling increases with working memory load, supporting a distributed account of working memory maintenance across these regions ([Kubska2021](../../raw/summaries/kubska2021_single_neuron_cognition.md)).
