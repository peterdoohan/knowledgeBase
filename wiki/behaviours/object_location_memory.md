# Object-Location Memory

## Current understanding

The hippocampus supports memory for both object identity and object location, with location serving as the primary organizing dimension of ensemble coding. Object identity is represented as a secondary cluster within location-based representations, supporting the cognitive map theory's proposal that the hippocampus maintains spatial maps with embedded non-spatial information.

## Key evidence

- Simultaneous recordings from rat hippocampal pyramidal neurons show that object location is the primary dimension of ensemble coding, with object identity represented as a secondary cluster within location-based representations. 60.0% of pyramidal cells showed significant firing-rate modulation by location, while only 16.6% showed modulation by object identity ([Manns 2009](../../raw/summaries/manns2009_cognitive_map_object_memory.md))

- Population-level analysis using nearest-neighbour classification correctly identified the same location for 72.7% of encounters (chance ~9.7%), while within-location object identification was 19.9% (chance ~7.7%). The same object encountered at different locations yielded no significant representational similarity (3.3% vs. chance 2.5%), indicating object identity is not represented independently of location ([Manns 2009](../../raw/summaries/manns2009_cognitive_map_object_memory.md))

- Two-factor repeated-measures ANOVA on representational similarity revealed significant main effects of location (F(1,3) = 102.45, P < 0.01), object identity (F(1,3) = 42.19, P < 0.01), and their interaction (F(1,3) = 75.96, P < 0.01). Same-object similarity was significantly higher than different-object similarity only within the same location, not across locations ([Manns 2009](../../raw/summaries/manns2009_cognitive_map_object_memory.md))

- Neural correlates of memory parallel behavioral effects: a neural repetition effect (shorter nearest-neighbour distances for object identity on second encounter vs. first; t(3) = 3.19, P < 0.05) and a neural relocation effect (longer nearest-neighbour distances for location when an object was moved; t(3) = 2.37, P < 0.05) were observed ([Manns 2009](../../raw/summaries/manns2009_cognitive_map_object_memory.md))

## History of ideas

The cognitive map hypothesis (O'Keefe and Nadel 1978) proposed that the hippocampus maintains a spatial map containing not just spatial layout but also the identities of objects encountered at specific locations. Early studies showed hippocampal neurons respond to specific objects or odors, but whether location and identity are equally weighted or hierarchically organized at the population level was unknown until ensemble recording studies.

## Open questions

- Whether the location-primary hierarchy arises from synaptic or circuit-level properties
- How object-location binding changes with learning and consolidation
- Whether ventral hippocampus shows different location-identity trade-offs
- The role of upstream areas (entorhinal, perirhinal cortex) in establishing the hierarchy
