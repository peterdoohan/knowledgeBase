# Tolman-Eichenbaum Machine

The Tolman-Eichenbaum Machine (TEM) is a computational framework that formalizes how the brain constructs and uses cognitive maps for generalisation across spatial and non-spatial domains, named after Edward Tolman (who proposed cognitive maps) and Howard Eichenbaum (who linked them to episodic memory).

## Key evidence

- The Tolman-Eichenbaum Machine framework operationalises factorisation as the separation of relational structure from sensory identity, enabling cross-graph generalisation. The abstract relational map in mPFC demonstrates this factorisation by representing graph position independent of specific stimulus identity ([Baram 2024](../../raw/summaries/baram2024_abstract_relational_map_consolidation.md))
