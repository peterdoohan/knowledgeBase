## Key evidence

- Working memory gating via corticostriatal circuits is a core mechanism of hierarchical control, modulating rostrocaudal activation patterns beyond abstraction level per se ([Badre 2018](../../raw/summaries/badre2018_frontal_cortex_hierarchical_control.md))
- Input and output gating models explain patterns of PFC activation; frontostriatal loops coordinate hierarchical level transitions ([Badre 2018](../../raw/summaries/badre2018_frontal_cortex_hierarchical_control.md))
- Corticostriatal-thalamic loops, especially output gating, coordinate network interactions; multiple topographically organized frontostriatal loops may support separate levels of the hierarchy ([Badre 2018](../../raw/summaries/badre2018_frontal_cortex_hierarchical_control.md))
- The core idea in hierarchical RL is that striatum disinhibits thalamic input to PFC to route contextual information, with nested loops supporting separate hierarchical levels ([Badre 2019](../../raw/summaries/badre2019_hierarchical_cognitive_control_lobes.md))

## History of ideas

- The corticostriatal gating model was developed by Frank, O'Reilly, Chatham and others as a mechanism for working memory updating in PFC.
- The model was extended to hierarchical control by Badre and colleagues, proposing that frontostriatal loops implement gating at different levels of a processing hierarchy.

## Open questions

- The neural implementation of working memory gating — whether corticocortical, corticostriatal-thalamic, or both — remains unresolved
- Which populations, timescales, and neuromodulators mediate cortico-striatal gating at a neural circuit level are not yet established
