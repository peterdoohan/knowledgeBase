## Key evidence

- Content-specific memory consolidation can be selectively modulated using closed-loop brain-machine interface methods, demonstrating that the memory content of hippocampal reactivation determines which memories are consolidated ([Gridchyn2020](../../raw/summaries/gridchyn2020_replay_selective_memory.md)).

- Bayesian population decoding combined with real-time optogenetic manipulation enables selective disruption of specific memory traces during sleep replay, sparing other simultaneously acquired memories ([Gridchyn2020](../../raw/summaries/gridchyn2020_replay_selective_memory.md)).
